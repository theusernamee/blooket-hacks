How to use 

 1. go to blooket (once in game or on dashboard
 2. hit crtl+shift+i
 3. select console
 4. type allow paste
 5. then paste code in (crtl+c for copy then paste with crtl+v)
 6. To use "get daily tokens" open blooket dashboard
 7. select play game
 8. then paste it into console without entering a game
 9. Warning! these will not work in solo mode
 10. If these have helped you pleaase consider leaving a star
